Robocraft JPのホームページへようこそ！

12/05/2025 - MaxSignal

ここは、日本で運営している非公式ロボクラサーバー、Robocraft JP のホームページです。

対象のクライアントは、Steamでの最終更新版(2021年版)です。

NGnius 氏の OpenJam プロジェクトにて開発されているサーバーソフトウェアを用いて運営しています。

普段は RC15JP という Discord サーバーでこのサーバーについて話しています。招待リンクは右上の Community 内にあります。

サーバーへの入り方

1. 右上の Register からアカウント登録をしてください。

2. ゲーム本体を Steam から入手するか、右上の Download から、アーカイブ(Windowsのみ)をダウンロードしてください。

3. 右上の OpenJam から Server List にアクセスして、Robocraft JP の欄の Connect をクリックしてコンフィグファイルをダウンロードしてください。

4. Steam の場合、ライブラリから Robocraft → プロパティ → インストール済みファイル → 参照 を選択します。(OSによって表示が違う場合があります。)

ファイルエクスプローラーが開くので、そこに先ほどのコンフィグファイルを設置してください。

アーカイブを入手した人は、アーカイブを展開したフォルダに Robocraft.exe 等が含まれていると思います。そこにコンフィグファイルを設置してください。

4. Steam の場合は通常通りプレイボタン、アーカイブの場合は Robocraft.exe を実行することで遊べます。

サポート情報

もし入れない場合は、RC15JP か、Twitter: MaxSignal, Discord: maxsignal2までお知らせください。

バグなどを見つけた場合は、OpenJam の Stoat(DiscordのようなSNS) または、Git に報告するか、RC15JP 等で教えてくれると助かります。
