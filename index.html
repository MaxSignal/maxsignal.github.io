日本語 / English

ここは、日本で運営している非公式ロボクラサーバー、Robocraft JP のホームページです。

対象のロボクラバージョンは、Steamでの最終更新版(2021年版)です。

NGnius 氏の OpenJam プロジェクトにて開発されているサーバーソフトウェアを用いて運営しています。

Discord サーバー: こちら

サーバーへの入り方

1. Register からアカウント登録をしてください。

2. ゲーム本体を Steam から入手するか、各OS向けのアーカイブ（Windows / Linux / macOS）をダウンロードしてください。

3. Server List にアクセスして、Robocraft JP の欄の Connect をクリックしてコンフィグファイルをダウンロードしてください。

4. Steam の場合、ライブラリから Robocraft → プロパティ → インストール済みファイル → 参照 を選択します。(OSによって表示が違う場合があります。)

ファイルエクスプローラーが開くので、そこに先ほどのコンフィグファイルを設置してください。

アーカイブを入手した人は、アーカイブを展開したフォルダに Robocraft.exe 等の実行ファイルが含まれていると思います。そこにコンフィグファイルを設置してください。

5. Steam の場合は通常通りプレイボタン、アーカイブの場合は Robocraft.exe 等を実行することで遊べます。

6. ログイン後、必ず ESC -> ENTER CODE から RCJP というプロモコードを適用してゲームを再起動ください。適用されていないとマルチプレイができません。

サポート情報

もし入れない場合は、Robocraft JP Discord か、Twitter: MaxSignal, Discord: maxsignalまでお知らせください。

バグなどを見つけた場合は、OpenJam の Stoat(DiscordのようなSNS) または、Git に報告するか、日本語での報告の場合は Srv の Git または Wiki のコメント欄にて教えてくれると助かります。
