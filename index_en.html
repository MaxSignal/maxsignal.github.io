日本語 / English

This is the home of Robocraft JP, a fan-run, unofficial Robocraft server based in Japan.

It targets the final Steam release of Robocraft (the 2021 version).

The server runs on software developed as part of NGnius's OpenJam project.

Discord server: here

There is also a Robocraft US server hosted in the United States. The steps to join it are different, so please check the Robocraft US Discord.

How to Join the Server

1. Register an account.

2. Get the game on Steam, or download the build for your OS (Windows / Linux / macOS).

3. Open the Server List, then click "Connect" on the Robocraft JP entry to download its config file.

4. On Steam, go to your Library → Robocraft → Properties → Installed Files → Browse (the exact wording may vary by OS).

A file explorer window will open — drop the config file you just downloaded into that folder.

If you are using an archive build, extract it and find the folder containing the executable (e.g. Robocraft.exe), then place the config file there.

5. Launch the game as usual: press Play on Steam, or run Robocraft.exe (or the equivalent) for the archive build.

6. After logging in, be sure to apply the promo code "RCJP" via ESC → ENTER CODE, then restart the game. Multiplayer will not work until the code is applied.

Support

If you cannot connect, reach out on the Robocraft JP Discord, on Twitter (MaxSignal), or on Discord (maxsignal).

Found a bug? Please report it in the OpenJam community — via Stoat (a Discord-like social platform) or on Git.
