Highlights of the main releases of the server software (OpenJam / rc-servers). For full details of each release, see the releases page.

v1.2.2 (2026-04-29)

A bug-fix patch covering clan avatar display, returning to the main menu in some game modes, leaving a clan as the last member, custom-game invites, and the TDM kill counter. It also adds Docker configuration and SI-unit support, and removes the 2000 CPU check when joining the multiplayer queue.

v1.2.1 (2026-04-06)

Adds friends, clans, and platoons, custom games, vehicle validation, and team-selection settings. Matchmaking now factors in platoons, and the delay before entering a match is reduced.

v1.1.0 (2026-02-14)

Adds an item shop, currencies (XP, tech points, robits, GC), promo codes, and a reward screen. Adds Factory (CRF) upload support with a dedicated website and API, along with many multiplayer fixes.

v1.0.0 (2025-12-06)

The first stable release, adding a public server list, inter-service communication (intercom), and a status endpoint for server information.

v0.4.0 (2025-10-04)

Implements most of multiplayer (Battle Arena, Team Death Match, and Pit) along with client-side NPCs.

v0.3.0 (2025-08-08)

First multiplayer support (1v1 elimination), proper account bans, and usernames that stay consistent across login methods.

v0.2.0 (2025-06-07)

Adds the CRF (a read-only archive), much more of garage/build mode, chat, and singleplayer with enemies.

v0.1.0 (2025-04-05)

The first release: login, edit mode, campaign (1 wave), and singleplayer, as a minimum viable version.
