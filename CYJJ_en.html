This page is about the server for the Chinese Robocraft client, "罗博造造" (CYJJ).

This client has features that no longer exist in the modern game, such as damage boost and health boost.

How to Join the Server

1. Get the game

From the official source

Tencent's servers still host a launcher that downloads the client installer — use this.

Once downloaded, run it. The launcher will start and download the client installer; when it finishes, click "立即安装" (Install Now) to launch the installer.

In the installer, keep clicking the button next to "取消" (Cancel). When you reach the screen for choosing the installation directory, pick wherever you want it and continue.

Finally, open the installation directory and confirm that Robocraft.exe is there. (From here on, we will refer to the folder containing this file as "CYJJ".)

From the archive

Download this and extract it.

Confirm that the extracted folder contains Robocraft.exe. (From here on, we will refer to this folder as "CYJJ".)

2. Patch the game to bypass Tencent's authentication and fix the GUI. Place this file in the CYJJ folder and run it.

3. Place servenvmulti.config, which tells the client which server to connect to, in the same folder.

4. Register an account on this page. (Warning: this Chinese client stores passwords in plain text, so never use a password you use anywhere else.)

5. Launch Robocraft.exe.

6. Log in with the username and password you just created.

Note that multiplayer on this server is normally set to 1v1. If you want to play with more people, please use a custom game.

If you need anything, reach out on Discord!
